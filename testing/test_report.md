# Отчет о тестировании

## Функциональное тестирование

| ID | Модуль | Тест-кейс | Входные данные | Ожидаемый результат | Фактический результат | Статус |
|----|--------|------------|----------------|---------------------|----------------------|---------|
| F1 | Бронирования | Создание нового бронирования | Имя: "Иван Иванов", Телефон: "+79161234567", Гости: 4, Дата: завтра, Время: 19:00 | Бронирование создается, появляется в таблице | ✅ Успешно | PASS |
| F2 | Бронирования | Редактирование бронирования | Изменение количества гостей с 4 на 6 | Данные обновляются в системе |  Успешно | PASS |
| F3 | Бронирования | Удаление бронирования | Выбор бронирования → Удалить | Бронирование удаляется из списка |  Успешно | PASS |
| F4 | Бронирования | Проверка конфликта времени | Бронирование на тот же стол и время | Система не позволяет создать конфликтующее бронирование | ✅ Успешно | PASS |
| F5 | Столы | Просмотр доступных столов | Выбор даты и количества гостей | Показываются только подходящие столы |  Успешно | PASS |
| F6 | Столы | Просмотр бронирований стола | Выбор стола → "Просмотреть бронирования" | Отображается список бронирований |  Успешно | PASS |
| F7 | Статистика | Обновление статистики | Добавление нового бронирования | Цифры в статистике обновляются |  Успешно | PASS |
| F8 | Валидация | Проверка пустых полей | Пустое имя клиента | Сообщение об ошибке |  Успешно | PASS |
| F9 | Валидация | Проверка даты | Прошедшая дата | Сообщение об ошибке |  Успешно | PASS |
| F10 | Фильтрация | Фильтр по дате | Выбор конкретной даты | Показываются только бронирования на эту дату |  Успешно | PASS |
