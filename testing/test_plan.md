# План тестирования системы бронирования кафе

## Цели тестирования
- Проверить функциональность основных модулей системы
- Оценить удобство пользовательского интерфейса
- Выявить возможные ошибки и узкие места
- Собрать обратную связь для улучшения системы

## Типы тестирования

### Функциональное тестирование
**Цель:** Проверить корректность работы основных функций системы
- Создание, редактирование, удаление бронирований
- Проверка доступности столов
- Фильтрация и поиск
- Работа с базой данных

### Пользовательское тестирование
**Цель:** Оценить удобство интерфейса и пользовательский опыт
- Навигация по интерфейсу
- Простота выполнения основных операций
- Восприятие визуального дизайна

## Инструменты и среда
- **ОС:** Windows 11
- **Java:** OpenJDK 24
- **JavaFX:** 25.0.1
- **БД:** PostgreSQL
- **Инструменты:** Ручное тестирование, Google Forms для опросов
