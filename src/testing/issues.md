# Детальный список проблем и предложений

## 4.1 Классификация проблем

### Критические ошибки (Блокирующие)
 **Не обнаружено**

### Значительные проблемы (Высокий приоритет)

#### ISSUE-001: Отсутствие поиска по клиентам
- **Описание:** Невозможно быстро найти бронирование по имени клиента
- **Воспроизведение:** Открыть вкладку "Бронирования" → попытаться найти конкретного клиента
- **Влияние:** Затрудняет работу при большом количестве бронирований
- **Предложение:** Добавить поле поиска над таблицей бронирований
- **Приоритет:** Высокий

#### ISSUE-002: Неинформативные сообщения о занятости столов
- **Описание:** При недоступности стола не показывается причина
- **Воспроизведение:** Выбрать дату/время когда стол занят → посмотреть доступные столы
- **Влияние:** Пользователь не понимает, почему стол недоступен
- **Предложение:** Добавить tooltip с информацией о существующем бронировании
- **Приоритет:** Высокий

### Улучшения пользовательского опыта (Средний приоритет)

#### ISSUE-003: Отсутствие иконок у кнопок
- **Описание:** Все кнопки текстовые, нет визуальных ориентиров
- **Влияние:** Замедляет восприятие интерфейса
- **Предложение:** Добавить иконки для основных действий 
- **Приоритет:** Средний

#### ISSUE-004: Нет подтверждения для опасных действий
- **Описание:** Удаление бронирования происходит без подтверждения
- **Влияние:** Возможно случайное удаление
- **Предложение:** Добавить диалог подтверждения для удаления
- **Приоритет:** Средний

### Функциональные улучшения (Низкий приоритет)

#### ISSUE-005: Отсутствие экспорта данных
- **Описание:** Невозможно выгрузить данные в Excel/PDF
- **Влияние:** Затрудняет создание отчетов
- **Предложение:** Реализовать функцию экспорта бронирований
- **Приоритет:** Низкий
