# Детальный план тестирования системы бронирования кафе

## 1.1 Общие сведения
- **Проект:** Система управления бронированиями кафе
- **Версия:** 1.0
- **Дата тестирования:** [21.11.2025]

## 1.2 Объем тестирования
### Включаемые компоненты:
- Модуль бронирований (CRUD операции)
- Модуль управления столами
- Система фильтрации и поиска
- Статистика и отчетность
- Валидация данных
- Работа с базой данных PostgreSQL

### Исключаемые компоненты:
- Интеграция с внешними системами
- Мобильная версия
- Система уведомлений

## 1.3 Стратегия тестирования
### Функциональное тестирование
**Метод:** Ручное тестирование по чек-листам
**Подход:** Снизу вверх (от модулей к интеграции)
**Критерии начала:** Все модули разработаны, БД настроена

### Пользовательское тестирование
**Метод:** Usability-тестирование с реальными пользователями
**Количество участников:** 5 человек
**Длительность сессии:** 15-20 минут на человека

## 1.4 Критерии качества
### Критерии приемки:
- 95% тест-кейсов пройдены успешно
- Критические ошибки отсутствуют
- Средняя оценка пользователей ≥ 8/10

### Критерии приостановки:
- Обнаружены критические ошибки, блокирующие работу
- Более 30% тест-кейсов провалены

## 1.5 Ресурсы
### Оборудование:
- ОС Windows 11
- 8 ГБ ОЗУ
- 256 ГБ SSD

### Программное обеспечение:
- OpenJDK 24
- JavaFX 25.0.1
- PostgreSQL 15
- IntelliJ IDEA

### Команда:
- Ведущий тестировщик: 1 чел.
- Тестировщики: 2 чел.
- Пользователи-тестировщики: 5 чел.

## 1.6 Расписание
| Этап | Длительность | Ответственный |
|------|--------------|---------------|
| Подготовка тестовой среды | 1 день | Ведущий тестировщик |
| Функциональное тестирование | 2 дня | Тестировщики |
| Пользовательское тестирование | 1 день | Все тестировщики |
| Анализ результатов | 1 день | Ведущий тестировщик |
| Составление отчетности | 1 день | Ведущий тестировщик |
